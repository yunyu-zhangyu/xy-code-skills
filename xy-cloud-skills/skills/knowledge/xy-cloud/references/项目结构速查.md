## 2. 业务模块内部结构

每个业务模块分为 api 和 biz 两个子模块：

```
platform-module-{name}/
├── platform-module-{name}-api/                  # 轻量 JAR（Feign 接口 + 常量）
│   └── src/main/java/cn/shutan/platform/module/{name}/
│       ├── api/                                  # Feign 客户端接口
│       ├── enums/                                # ErrorCodeConstants, ApiConstants
│       └── dto/                                  # 响应 DTO
│
└── platform-module-{name}-biz/                  # 可执行 JAR（业务实现）
    └── src/main/java/cn/shutan/platform/module/{name}/
        ├── controller/admin/{domain}/            # Controller
        │   └── vo/                               # SaveReqVO/PageReqVO/RespVO
        ├── service/{domain}/                     # Service 接口 + Impl
        ├── dal/
        │   ├── dataobject/{domain}/              # DO 实体
        │   └── mysql/{domain}/                   # MyBatis-Plus Mapper
        ├── convert/{domain}/                     # MapStruct 转换器
        ├── config/                               # 模块级配置
        └── job/                                  # XXL-Job（按需）
```
