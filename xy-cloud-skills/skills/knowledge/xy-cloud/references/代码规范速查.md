|------|------|------|---------|
| 创建 | `@PostMapping` | `/create` | `CommonResult<Long>` |

| 删除 | `@DeleteMapping` | `/delete` | `CommonResult<Boolean>` |
| 单条查询 | `@GetMapping` | `/get` | `CommonResult<RespVO>` |
| 列表查询 | `@GetMapping` | `/list` | `CommonResult<List<RespVO>>` |
| 分页查询 | `@GetMapping` | `/page` | `CommonResult<PageResult<RespVO>>` |
| 导出 | `@PostMapping` | `/export` | `void`（HttpServletResponse） |

```java
@Tag(name = "管理后台 - XXX")
@RestController
@RequestMapping("/{module}/{resource}")
@Validated
public class XxxController {
    @Resource
    private XxxService xxxService;

    @PostMapping("/create")
    @Operation(summary = "创建XXX")
    @PreAuthorize("@ss.hasPermission('{module}:{resource}:create')")
    public CommonResult<Long> createXxx(@Valid @RequestBody XxxSaveReqVO reqVO) {
        return success(xxxService.createXxx(reqVO));
    }
}
```

## 4. Mapper 层

继承 `BaseMapperX<XxxDO>`，使用 `LambdaQueryWrapperX` 构建条件：

来源：`platform-module-infra/platform-module-infra-biz/src/main/java/cn/shutan/platform/module/infra/dal/mysql/config/ConfigMapper.java`

```java
@Mapper
public interface XxxMapper extends BaseMapperX<XxxDO> {
    default PageResult<XxxDO> selectPage(XxxPageReqVO reqVO) {
        return selectPage(reqVO, new LambdaQueryWrapperX<XxxDO>()
                .likeIfPresent(XxxDO::getField, reqVO.getField())
                .eqIfPresent(XxxDO::getStatus, reqVO.getStatus())
                .betweenIfPresent(XxxDO::getCreateTime, reqVO.getCreateTime())
                .orderByDesc(XxxDO::getId));
    }
}
```

Mapper 支持批量操作：`insertBatch` / `updateBatch` / `delete` 已在 BaseMapperX 中提供。

## 6. DO 继承体系

来源：`platform-framework/platform-spring-boot-starter-mybatis/src/main/java/cn/shutan/platform/framework/mybatis/core/dataobject/BaseDO.java`

- `BaseDO`: `createTime` / `updateTime` / `creator` / `updater` / `deleted`（逻辑删除）
- `TenantBaseDO`: 继承 BaseDO，追加 `tenantId` 字段

字段自动填充（`@TableField(fill = FieldFill.INSERT)`），逻辑删除（`@TableLogic`），**无需手动赋值**。

## 9. Mapper XML 规范

来源：`platform-module-infra/platform-module-infra-biz/src/main/resources/mapper/`

- 文件位置：`src/main/resources/mapper/{Entity}Mapper.xml`
- 使用 `resultType`（非 resultMap），值为 DO 全限定名
- CRUD 操作由 BaseMapperX 处理，XML 仅用于多表关联查询
- 所有查询注意逻辑删除字段 `deleted = 0`
