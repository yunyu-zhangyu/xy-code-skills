# xy-cloud 可复用组件索引

> 根包：`cn.shutan.platform` | M0 生成时间：2026-06-12

## 1. 通用基础类 (platform-common)

| 类名 | 包路径 | 用途 |
|------|--------|------|
| `CommonResult<T>` | `pojo/CommonResult.java` | 统一 API 响应封装 |
| `PageResult<T>` | `pojo/PageResult.java` | 分页结果封装 |
| `PageParam` | `pojo/PageParam.java` | 分页请求基类 (pageNo=1, pageSize=10) |
| `BeanUtils` | `util/object/BeanUtils.java` | 对象转换 (DO↔VO) |
| `CollectionUtils` | `util/collection/CollectionUtils.java` | 集合操作工具 |
| `JsonUtils` | `util/json/JsonUtils.java` | JSON 处理 (Jackson) |
| `SpringUtils` | `util/spring/SpringUtils.java` | Spring 容器工具 |
| `ServletUtils` | `util/servlet/ServletUtils.java` | Web 操作工具 |
| `StrUtils` | `util/string/StrUtils.java` | 字符串工具 |
| `LocalDateTimeUtils` | `util/date/LocalDateTimeUtils.java` | 日期工具 |
| `CacheUtils` | `util/cache/CacheUtils.java` | 缓存工具 (buildAsyncReloadingCache) |
| `TracerUtils` | `util/monitor/TracerUtils.java` | 链路追踪 (getTraceId) |
| `ServiceExceptionUtil` | `exception/util/ServiceExceptionUtil.java` | 业务异常抛出 |
| `ErrorCode` | `pojo/ErrorCode.java` | 错误码定义 |

## 2. MyBatis 框架 (mybatis-starter)

| 组件 | 用途 | 位置 |
|------|------|------|
| `BaseDO` | 实体基类（时间/创建者/逻辑删除） | `...mybatis/core/dataobject/BaseDO.java` |
| `TenantBaseDO` | 多租户实体基类（继承 BaseDO + tenantId） | `...mybatis/core/dataobject/TenantBaseDO.java` |
| `BaseMapperX<T>` | Mapper 基础接口（CRUD + 批量操作） | `...mybatis/core/mapper/BaseMapperX.java` |
| `LambdaQueryWrapperX<T>` | 增强查询构造器（ifPresent 方法） | `...mybatis/core/query/LambdaQueryWrapperX.java` |
| `MPJLambdaWrapperX<T>` | 多表 Join 查询包装器 | `...mybatis/core/query/MPJLambdaWrapperX.java` |

## 3. Web 框架 (web-starter)

| 组件 | 用途 |
|------|------|
| 全局异常处理器 | 自动转换 ServiceException → CommonResult |
| 全局响应体处理器 | Controller 返回值自动包装 |
| `@Idempotent` | 防重复提交 (timeout/message) |
| `@RateLimiter` | 接口限流 (滑动窗口) |

## 4. 安全框架 (security-starter)

| 组件 | 用途 | 来源 |
|------|------|------|
| `SecurityFrameworkUtils.getLoginUserId()` | 获取当前登录用户 ID | 源码参考 |
| `@PreAuthorize("@ss.hasPermission('xxx:xxx:xxx')")` | 权限控制注解 | Controller 普遍使用 |

## 5. 多租户 (tenant-starter)

| 组件 | 用途 |

| `@TenantIgnore` | 忽略租户过滤 |

## 6. 数据权限 (data-permission)

| 组件 | 用途 |
|------|------|
| `@DataPermission(enable=false)` | 关闭数据权限过滤 |

## 7. 其他 Starter

| Starter | 能力 | Key 组件 |
|---------|------|---------|
| excel | Excel 导入导出 | `ExcelUtils`, `@Excel` 注解 |
| mq | 消息队列 | Redis Stream / RabbitMQ / RocketMQ / Kafka |
| job | 定时任务 | XXL-Job |
| websocket | WebSocket 推送 | WebSocketSenderApi (infra-api) |
| protection | 防重复/限流/签名 | `@Idempotent`, `@RateLimiter`, `@ApiSignature` |

## 8. Infra 模块 API

| Feign 接口 | 方法 | 用途 |

| `ConfigApi` | `getConfigValueByKey(key)` | 获取配置值 |
