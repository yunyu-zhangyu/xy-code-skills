# xy-cloud 缓存使用规范速查

## 三种缓存模式对比

| 模式 | 存储 | 一致性 | 适用场景 |
|------|------|--------|----------|
| Guava Local Cache | JVM 堆内 | 进程级，最终一致 | 高频读、可容忍节点间不一致 |
| Spring Cache (Redis) | Redis | 分布式一致 | 标准 CRUD、需跨节点一致 |
| 程序化 Redis DAO | Redis | 强一致（原子操作） | 动态 TTL、复杂键模式 |

## Guava Local Cache (CacheUtils)

**类**: `cn.shutan.platform.framework.common.util.cache.CacheUtils`

```java
// 异步刷新：过期时一个线程阻塞加载，其他线程返回旧值
LoadingCache<K, V> cache = CacheUtils.buildAsyncReloadingCache(
    Duration.ofMinutes(1), new CacheLoader<K, V>() { ... });

// 同步刷新
LoadingCache<K, V> cache = CacheUtils.buildCache(
    Duration.ofMinutes(1), new CacheLoader<K, V>() { ... });
```

| 使用位置 | TTL |
|----------|-----|
| SecurityFrameworkServiceImpl（权限/角色） | 1 min |
| TenantFrameworkServiceImpl（租户验证） | 1 min |
| DictFrameworkUtils（字典数据） | 1 min |
| SocialClientServiceImpl（WxMpService 实例） | 10 s |

## Spring Cache (Redis)

### 自动配置

| 配置类 | 功能 |
|--------|------|
| PlatformRedisAutoConfiguration | RedisTemplate<String,Object> + Jackson JSON + JavaTimeModule |
| PlatformCacheAutoConfiguration | @EnableCaching + 单冒号分隔符 + TimeoutRedisCacheManager |
| TimeoutRedisCacheManager | 缓存级 TTL（cacheName#ttl 格式） |
| TenantRedisCacheManager | 自动追加 :{tenantId} 做租户隔离 |

### TTL 配置格式

在 `@Cacheable(cacheNames = "role#10m")` 中通过 cacheName 指定：

| 后缀 | 单位 | 示例 |
|------|------|------|
| d | 天 | `cacheName#1d` |
| h | 小时 | `cacheName#2h` |
| m | 分钟 | `cacheName#30m` |
| s | 秒 | `cacheName#90s`（默认） |

### RedisKeyConstants 模式

```java
// interface 常量 — 统一管理缓存名称
public interface RedisKeyConstants {
    String ROLE = "role";
    String DEPT_CHILDREN_ID_LIST = "dept_children_ids";
    String OAUTH_CLIENT = "oauth_client";
    // ...（值 = 纯英文小写 + 下划线，不加冒号）
}
```

### @Cacheable 读取

```java
// 必需属性: value/cacheNames, key（SpEL）
// 推荐: unless = "#result == null"（防缓存穿透）
@Cacheable(value = RedisKeyConstants.ROLE, key = "#id", unless = "#result == null")
public RoleDO getFromCache(Long id) { return roleMapper.selectById(id); }
```

### @CacheEvict 失效（写失效策略）

```java
// 单条失效
@CacheEvict(value = RedisKeyConstants.ROLE, key = "#updateReqVO.id")
public void update(RoleSaveReqVO updateReqVO) { ... }

// 整表失效
@CacheEvict(cacheNames = RedisKeyConstants.OAUTH_CLIENT, allEntries = true)
public void updateOAuth2Client(OAuth2ClientSaveReqVO reqVO) { ... }

// 条件失效
@CacheEvict(value = RedisKeyConstants.PERMISSION_MENU_ID_LIST,
        key = "#createReqVO.permission",
        condition = "#createReqVO.permission != null")
public Long createMenu(MenuSaveVO createReqVO) { ... }
```

### @Caching 批量失效

```java
@Caching(evict = {
    @CacheEvict(value = RedisKeyConstants.MENU_ROLE_ID_LIST, allEntries = true),
    @CacheEvict(value = RedisKeyConstants.PERMISSION_MENU_ID_LIST, allEntries = true)
})
public void assignRoleMenu(Long roleId, Set<Long> menuIds) { ... }
```

## 程序化 Redis DAO 模板

```java
@Component
public class XxxRedisDAO {
    private final StringRedisTemplate stringRedisTemplate;
    private static final String KEY_PREFIX = "xxx:";

    public XxxRedisDAO(StringRedisTemplate stringRedisTemplate) {
        this.stringRedisTemplate = stringRedisTemplate;
    }

    private static String formatKey(String suffix) { return KEY_PREFIX + suffix; }

    public void set(String id, String value, long timeout, TimeUnit unit) {
        stringRedisTemplate.opsForValue().set(formatKey(id), value, timeout, unit);
    }

    public String get(String id) {
        return stringRedisTemplate.opsForValue().get(formatKey(id));
    }

    public void delete(String id) {
        stringRedisTemplate.delete(formatKey(id));
    }
}
```

### 5 个内置 DAO

| DAO | Key 前缀 | 特点 |
|-----|----------|------|
| OAuth2AccessTokenRedisDAO | `oauth2_access_token:` | 动态 TTL（取令牌过期时间） |
| IdempotentRedisDAO | `idempotent:` | SET NX EX，由 @Idempotent 使用 |
| RateLimiterRedisDAO | `rate_limiter:` | Redisson RRateLimiter（令牌桶） |
| ApiSignatureRedisDAO | `api_signature_*:` | nonce 防重放 + API key 存储 |
| RedisCaptchaServiceImpl | — | 验证码存储带 TTL |

## CRUD 缓存完整模式

```java
@Service
public class XxxServiceImpl implements XxxService {
    // CREATE — 不涉及缓存
    public Long create(XxxSaveReqVO reqVO) { ... }

    // UPDATE — 失效单条
    @CacheEvict(value = RedisKeyConstants.XXX, key = "#updateReqVO.id")
    public void update(XxxSaveReqVO updateReqVO) { ... }

    // DELETE — 失效单条
    @CacheEvict(value = RedisKeyConstants.XXX, key = "#id")
    public void delete(Long id) { ... }

    // GET — 读缓存回源
    @Cacheable(value = RedisKeyConstants.XXX, key = "#id", unless = "#result == null")
    public XxxDO getFromCache(Long id) { return mapper.selectById(id); }
}
```

## Key 命名规范

| 分类 | 模式 | 示例 |
|------|------|------|
| 业务缓存 | `{cacheName}:{entityId}` | `role:123` |
| 多租户 | `{cacheName}:{entityId}:{tenantId}` | `role:123:1` |
| 幂等 | `idempotent:{key}` | `idempotent:order:456` |
| 限流 | `rate_limiter:{key}` | `rate_limiter:ip:10.0.0.1` |
| 锁 | `lock4j:{key}` | `lock4j:order:456` |
