## 1. 字段类型对照

### Java ↔ MySQL 类型映射

| Java 类型 | MySQL 类型 | 长度/精度 | 说明 |
|-----------|-----------|-----------|------|
| `Long` | `bigint` | 20 | 主键 ID、金额（分） |
| `Integer` | `tinyint` / `int` | 3-10 | 状态码、排序号（**system/infra 状态码必须用 Integer**；eos 模块例外：varchar(8)） |
| `String` | `varchar` | 50-500 | 名称、编码、备注 |
| `String` | `char` | 1-2 | 固定长度标识（Y/N） |
| `BigDecimal` | `decimal` | 18,6 | 金额、比率、单价 |
| `LocalDateTime` | `datetime` | — | 日期时间 |
| `LocalDate` | `date` | — | 日期 |
| `Boolean` | `bit(1)` | 1 | 布尔值（0/1），如 deleted、visible |
| `byte[]` | `longblob` | — | 文件二进制内容 |

### 常用字段长度约定

| 场景 | 类型 | 长度 | 示例 |
|------|------|------|------|
| 名称 | varchar | 50 | 部门名、规则名 |
| 编码 | varchar | 50 | 字典编码（需 UNIQUE） |
| 备注 | varchar | 500 | 通用备注 |
| 手机号 | varchar | 11 | 用户手机 |
| 邮箱 | varchar | 100 | 用户邮箱 |
| URL | varchar | 500 | 文件路径 |
| JSON 配置 | varchar | 2000 | 扩展配置 JSON |

## 2. 状态枚举定义

### 通用开关状态

| 值 | 含义 | 说明 |
|----|------|------|
| 0 | 正常 / 启用 | 默认可操作状态 |
| 1 | 停用 / 禁用 | 管理员手动禁用 |
| -1 | 删除 | 逻辑删除 |

### 通用布尔标识

| 值 | 含义 |
|----|------|
| 0 | 否 / 关闭 / false |
| 1 | 是 / 开启 / true |

### 通用枚举参考

| 枚举 | 值 | 含义 |
|------|----|------|
| CommonStatusEnum | 0/1 | 通用启用/停用 |
| MenuTypeEnum | 1/2/3 | 菜单类型（目录/菜单/按钮） |
| SexEnum | 1/2/0 | 男/女/未知 |

### 状态字段类型规范

| 模块 | Java 类型 | 数据库类型 | 说明 |
|------|----------|-----------|------|
| system / infra | `Integer` | `tinyint(3)` | 0=启用, 1=停用。**禁止用 String** |
| eos / demand-response | `String` | `varchar(8)` | 字符串编码（如 '01', '0201'） |

> ⚠️ **为什么 system/infra 禁止用 String**：`LambdaQueryWrapperX.eqIfPresent(DO::getStatus, reqVO.getStatus())` 中，若 DO 的 status 是 String 而 reqVO 的 status 是 Integer，eqIfPresent 会静默跳过该条件（类型不匹配），查询返回全量结果而非按状态过滤。项目已有不一致（AdminUserDO 用 Integer，DeptDO 用 String），新实体必须统一为 Integer。

## 5. 命名规范速查

| 项目 | 规范 | 示例 |
|------|------|------|
| Java 类名 | PascalCase | `EnergyPriceController` |
| Java 方法名 | camelCase | `createEnergyPrice` |
| Java 包名 | 全小写 | `controller.admin.energyprice` |
| 数据库表名 | 下划线 | `eos_energy_price` |
| 数据库字段 | 下划线 | `price_value` |
| REST 路径 | kebab-case | `/eos/energy-price/page` |
| 权限标识 | `{module}:{resource}:{action}` | `eos:energy-price:create` |
| 文件名前缀 | `{来源前缀}-{实体}-{类型}` | `req-energy-price.md` |

### 文件名前缀规则

| 来源 | 前缀 |
|------|------|
| M1 需求澄清 | `req-` |
| M2 数据模型 | `dm-` |
| M3 组件查询 | `cq-` |
| M4 代码生成 | `gen-` |
| M5 边界分析 | `bd-` |
| M6 测试生成 | `tc-` |
| M7 测试执行 | `tr-` |
